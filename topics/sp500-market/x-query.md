# S&P 500 Market X Query Notes

This topic intentionally does not use a default X query.

Broad market X search is usually too noisy for a public template: stock-pick spam, technical chart posts, engagement bait, single-name hype, and recycled market-wrap links tend to overwhelm usable signal.

The live topic should lean on:

- `marketData.symbol = "SPY"`
- Google News context for broad catalysts
- Primary macro/policy sources when available

If a user wants to add X later, start with a narrow trusted-account query only and avoid broad ticker search.
